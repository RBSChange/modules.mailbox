Introduction

Le module « Boîte de réception » permet de stocker l’ensemble des formulaires envoyés par les internautes via le site.

Pour visualiser le détail d'un formulaire :

  • Sélectionner un formulaire dans la liste en cliquant sur son intitulé
  • Double-cliquer sur le formulaire

La fenêtre des propriétés avec le détail du formulaire s'affiche en bas de page.
